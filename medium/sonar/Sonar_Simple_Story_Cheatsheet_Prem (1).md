## 📖 The Story

**Once, humans wrote all computer code by hand** — slowly, one line at a time. Code is the instructions that make apps, games, and websites work.

**Then robot helpers showed up** 🦾 — AI agents like Claude, Cursor, and Copilot. Tell them "build me this," and *poof* — code in minutes. What took a day now takes minutes. Everyone cheered. 🎉

**But there was a problem.** 😬 The robots were fast but messy. Like a robot building a LEGO tower super fast — but some blocks are crooked, and the tower might wobble and fall later.

**Scientists at Carnegie Mellon proved it:**
- At first → faster, great 🚀
- After **3 months** → the mess piles up
- The code gets so tangled it actually slows everyone *down* 🐌
- (Warnings **+30%**, complexity **+41%**, and it *stays*)

The robots don't notice the mess — because someone *else* has to clean it up.

**Enter the hero: Sonar!** 🦸 A super-smart **inspector** that checks the robot's work:
> "✅ This part's good! ❌ This part's broken — fix it! ⚠️ This part isn't safe!"

And it checks **right away** — while the robot is still building — not days later when it's too late. Like a teacher beside you saying "check that one again" *before* you finish.

## 🔁 The AC/DC Cycle (4 steps)

| Step | Emoji | What happens |
|------|-------|--------------|
| **Guide** | 🗺️ | Before building, Sonar tells the robot "here are the rules of *this* house" |
| **Generate** | ✍️ | The robot writes the code |
| **Verify** | 🔍 | Sonar inspects it — good? safe? tidy? |
| **Solve** | 🔧 | If broken, a fix-it robot repairs it — and Sonar double-checks the fix really worked before anyone uses it |

**The big idea in one sentence:**
> Robots write code fast but messy. Sonar is the careful inspector that catches and cleans the messy parts, so the things we build don't fall over. 🧹✨

**The proof it works:** Companies using Sonar have **44% fewer "everything broke!" moments.** Nvidia, Goldman Sachs, Ford all trust it. 🏆
