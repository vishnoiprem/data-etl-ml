# Real-Time Financial Reporting Platform

A complete, runnable data pipeline for financial reporting automation across multiple countries.

## Features

- **End-to-End Pipeline**: From data extraction to reporting
- **Multi-Country Support**: Thailand (TH), Vietnam (VN), Indonesia (ID)
- **Simulated Data**: Generate realistic financial transactions
- **Data Quality Monitoring**: Automated checks and anomaly detection
- **Interactive Dashboard**: HTML dashboard with Plotly charts
- **Excel Export**: Business-ready reports in Excel format
- **Complete Test Suite**: Unit and integration tests
